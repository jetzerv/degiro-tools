# DEGIRO-TOOLS
Tools that help you extend your insights into your DEGIRO portfolio

## Ideas
- **Yahoo Finance Converter**: Convert DEGIRO Transactions into Yahoo Finance importable csv
- **YoY Performance tracker**: Track your Year over Year performance based on DEGIRO transaction data
- **Compare Portfolio vs Index**: Compare your portfolio performance against a benchmark index like S&P 500 or MSCI World
